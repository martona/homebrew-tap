# Martona Homebrew Tap

Homebrew formulae for martona macOS stuff.

## Install

```sh
brew tap martona/tap
brew install yo
```

Or install `yo` directly:

```sh
brew install martona/tap/yo
```

After installation, run:

```sh
yo --setup
```

The `yo` formula installs the signed and notarized macOS release binary from
[`martona/yo`](https://github.com/martona/yo).
